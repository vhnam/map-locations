# map-locations

This is a sample about [leaflet](https://leafletjs.com/) and [turf](https://turfjs.org/).

## Installation

```
yarn install
```

## Start

```
yarn start
```

## Note

You will confused about the position of lattitude and longitude in `leaflet` and `turf`

```
turf: [lng, lat]
leaflet: [lat, lng]
```
